The files in this directory are used within UIOPS to translate OAP data from native format 
to netCDF files readable by UIOPS. Currently supported files inlcude the following:
  SPEC 2DS
  PMS 2D files in standard NCAR 2D format (includes 2DC, 2DP, and fast2DC)
  DMT formats for CIP and PIP
  OAP data from SEA data systems
